# Contributing

Thanks for helping grow the **Awesome MPP / x402** list and the
[mpp.best](https://mpp.best) directory of AI payment & x402 projects.

## How to add a project

There are two ways:

1. **Easiest — submit on the website:** fill out the short form at
   **[mpp.best/submit](https://mpp.best/submit)**. New entries flow into both
   the directory and this list automatically.

2. **Open a Pull Request:** add your project to the relevant section of
   `README.md` using this format:

   ```markdown
   | [owner/repo](https://github.com/owner/repo) | ⭐stars | Language | One-line description |
   ```

## Inclusion criteria

To keep the list genuinely useful (and rank-worthy), a project should be:

- **On topic** — related to x402, AI agent payments, USDC / stablecoin
  micropayments, MCP payments, or agentic commerce.
- **Real & maintained** — a working repo, package, or tool (not a placeholder).
- **Clearly described** — a one-line description of what it does.

Off-topic, abandoned, or spam entries may be removed to preserve quality.

## Add the badge

If your project is listed, feel free to add this badge to your README:

```markdown
[![Listed on mpp.best](https://img.shields.io/badge/Listed%20on-mpp.best-D97A5E)](https://mpp.best)
```
